# Stock Signal Bot v8.0

Ushbu bot aksiyalarni tahlil qilish, grafiklar chizish va AI orqali strategiya tavsiyalarini berish uchun mo'ljallangan.

## ✨ Imkoniyatlar
- 📊 **Real-vaqt grafiklar:** Matplotlib orqali texnik tahlil chizmalari.
- 🤖 **AI Analiz:** OpenAI yordamida bozor holatini tahlil qilish.
- 🕋 **Halollik tekshiruvi:** Musaffa va Akooda platformalaridan ma'lumot olish.
- 📰 **Yangiliklar:** Eng so'nggi fond bozori yangiliklari.

## 🛠 O'rnatish
1. Kutubxonalarni o'rnating:
   `pip install -r requirements.txt`
2. `.env` fayliga o'z API kalitlaringizni (TELEGRAM_TOKEN, POLYGON_KEY, OPENAI_API_KEY) joylashtiring.
3. Botni ishga tushiring:
   `python bot.py`
